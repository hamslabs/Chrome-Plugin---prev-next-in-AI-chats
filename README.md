## Prompt Navigator (Chrome/Brave extension)

### What it does
Adds keyboard shortcuts to jump to the next/previous **user prompt** in common AI chat UIs.

### Install (developer mode)
1. Open `chrome://extensions` (or `brave://extensions`).
2. Enable **Developer mode**.
3. Click **Load unpacked** and select this folder.

### Default shortcuts
- Next prompt: macOS `Option+J` (others: `Alt+Shift+J`)
- Previous prompt: macOS `Option+K` (others: `Alt+Shift+K`)

You can change these at `chrome://extensions/shortcuts`.

### Supported sites
- ChatGPT (`chat.openai.com`)
- Claude (`claude.ai`)
